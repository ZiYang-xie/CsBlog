Summary

  1. Accepts a volume of size $W_1 H_1 D_1$
  2. Four Hyperparas
    • Number of filters $K$
    • spatial extent $F$
    • stride $S$
    • zero padding amount $P$
  3. Produces a volume of size $W_2 H_2 D_2$
    • $W_2 = (W_1 - F + 2P)/S + 1$
    • $H_2 = (H_1 - F + 2P)/S + 1$
    • $D_2 = K$ Depth keeps the same

Common Settings

K = (powers of 2)

  • F = 3, S = 1, P =1
  • F = 5, S = 1, P =2
  • F = 1, S = 1, P = 0
