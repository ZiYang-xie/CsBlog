An Observation on Generalization - Ilya

An Observation on Generalization - Ilya

Presenter: Ilya Sutskever

Affiliation: OpenAI

Link: https://www.youtube.com/watch?v=AKMuA_TVz3A&t=23s

1. What is Learning and Why it works?

Supervised Learning: Is well-understood

if Distribution(Training Data) = Distribution(Training Data)

  • Low Test Error = Low Training Loss + Large Training Data

Unsupervised Learning: Mysterious

  • Did not show up at small scale
  • Denosing autoencoder: BERT and diffusion

Optimize on one object → Expected high performance on another

Unsupervised Learning via Distribution Matching

Given data $X$ and $Y$ → Find the function $F$ so that

$Distribution(F(X)) \sim Distribution(Y)$

Q1. How to find that uniform mapping to map the distribution into the same space?

Compression to the rescue

For a good compression algorithm $C(data)$

It should satisfy:

  • Compress know the shared information with in $X$ and $Y$

As a result, the compressor will know the function $F$ that could map the $X$ to the same (or at least close) distribution with $Y$

A1: Find a best compression algorithm $C(data)$, it will know $F$

Q2: How to reach lowest regret in self-supervised learning?

  • Find a program that implement the best compressor $C$

So how to formalize the program?

Kolmogorov complexity

  • K(X) = length of shortest program that output X
  • If $C$ is a computable compressor, then

A2: Find the Kolmogorov compressor helps!

Q3: How to find that Kolmogorov compressor?

Issue: It is undecidable and uncomputable

Solution: A deep NN/transformer is a parallel little computers

(NN can simulate program, that make Kolmogorov compressor computable)

SGD = searching over the program / circuit

A3: Using NN with SGD to search the Kolmogorov function

Given data $X$ and $Y$, It’s the unsupervised learning that try to predict $Y$ from data $X$!

This will extract all the useful ‘value’ out of $X$ for predicting $Y$

  • Just compress everything
  • Concat all the data → $K(X,Y)$
  • Then we can still find a good $K(Y|X)$

SGD over big NNs ~ Our program search

GPT can be derived by straight-forward reasoning about conditional distribution of text

Q: Does this pattern also applies to vision?

Next pixel prediction

A: Yes, It Kind of work

However, the compression theory does not explain why representation are nice and linearly separable.

  • AutoRegressive model seem to have better representations than BERT
